## 商店信息模板

### 扩展名称
Card Tab - 我的导航

### 简短描述（最多 132 字符）
自定义新标签页导航，支持分类管理、拖拽排序、深色模式，数据本地存储。

### 详细描述
Card Tab 是一款简洁美观的新标签页导航扩展，让您快速访问常用网站。

功能特性：
• 自定义分类和链接管理
• 拖拽排序卡片
• 深色/浅色模式自动切换
• 本站搜索 + 多搜索引擎
• 数据导入/导出（JSON 格式）
• APP 视图/卡片视图切换
• 响应式设计，适配各种屏幕

隐私优先：
所有数据存储在您的浏览器本地，不会上传到任何服务器。

### 类别
- Chrome: 生产工具
- Firefox: 标签页
- Edge: 生产力

### 语言
中文（简体）

## 发布步骤

### Chrome Web Store
1. 访问 https://chrome.google.com/webstore/devconsole
2. 支付 $5 注册费
3. 将 `extension` 文件夹打包为 ZIP
4. 上传并填写信息

### Firefox Add-ons
1. 访问 https://addons.mozilla.org/developers/
2. 免费注册
3. 上传 ZIP 包

### Edge Add-ons
1. 访问 https://partner.microsoft.com/dashboard/microsoftedge
2. 免费注册
3. 上传 ZIP 包
